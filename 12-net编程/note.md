# 网络编程
- 网络：
- 网络协议：一套信息交流规则
- 网络模型
    - 理论模型 - 七层
        - 物理层（可触摸的配件及设备）
        - 数据链路层（信号传递）
        - 网络层
        - 传输层
        - 会话层
        - 表示层
        - 应用层
    - 实际应用模型 - 五层
        - 物理层
        - 链路层
        - 网络
        - 传输层
        - 应用层
- 每一层都有相应的协议负责交换信息或协同工作
- TCP/IP 协议族
- IP 地址：负责在网络上唯一定位一个机器
    - IP 地址分 A B C D E 五类
    - 室友四个数字段组成，每个数字段的取值是 0 - 255
