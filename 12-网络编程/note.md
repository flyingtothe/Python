# 网络编程
- 网络：
- 网络协议：一套信息交流规则
- 网络模型
    - 理论模型 - 七层
        - 物理层（可触摸的配件及设备）
        - 数据链路层（信号传递）
        - 网络层
        - 传输层
        - 会话层
        - 表示层
        - 应用层
    - 实际应用模型 - 五层
        - 物理层
        - 链路层
        - 网络
        - 传输层
        - 应用层
- 每一层都有相应的协议负责交换信息或协同工作
- TCP/IP 协议族
- IP 地址：负责在网络上唯一定位一个机器
    - IP 地址分 A B C D E 五类
    - 室友四个数字段组成，每个数字段的取值是 0 - 255
    - 192.168.xxx.xxx;局域网ip
    - 127.0.0.1：本机
    - IPV4, IPV6
- 端口
    - 范围：0 - 65535
        - 知名端口：0 - 1023
        - 非知名端口：1024 -
 
# TCP/UDP协议
- UDP：非安全的，不面向连接的传输
    - 安全性差
    - 大小限制64kb
    - 传递没有顺序
    - 速度快
- TCP
    - 基于连接的通信
- SOCKET 编程
